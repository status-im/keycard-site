Secure, Contactless, Open Source Hardwallet

Keycard is a new type of smartcard built with an open API for simple integration with crypto wallets and hardware. Designed to safely send, store, and receive cryptocurrencies in a seamless contactless experience.

Get my Keycard

Creating A Secure Crypto Ecosystem

Compatible with both mobile and desktop wallets, keycard creates a safe and trustworthy crypto ecosystem for users. It can be loaded with cryptocurrency to be used as a debit card and also integrate with point of sale systems for easy retail crypto transactions.

Do more with Keycard

Compatible with all Major Cryptocurrencies

An HD BIP-32 wallet which safely stores Ethereum, ERC20 tokens, Bitcoin, Bitcoin Cash, Litecoin, Ripple, and any currency based on ECDSA signatures.

Contactless Transaction Verification

With NFC, you can securely sign and verify transactions & payments by tapping Keycard on your mobile device.

Tap-To-Launch

Users can launch a designated app or a URL simply by tapping their card on their phone.

Inexpensive und Familiar Design

Designed similar to a credit/debit card, Smartcards are an inexpensive, simple, and secure way to onboard new users.

Works with Desktop Wallets Too

Keycard can easily be used with a desktop application with any USB-contact reader.

Built for Security

Keycard is designed to safely store your private keys offline in an air-gapped environment. They never leave the secure element of the card and are always kept offline.

See the Docs

Keycard hardware security is compliant with the most stringent standard (Common Criteria EAL5+) . No one will be able to access private keys even if the card is stolen.

Generate secrets with the built in True Random Number Generator.

An Open Source API

Both Keycard and the Keycard API are entirely open source and allow any developer to integrate with mobile apps, desktop wallets, point of sale retail devices and even create their own smartcards.

Easy to Use Java (Android & Desktop) and Go SDK
High-level classes abstract away the most tedious tasks including Secure Channel, APDU parsing, serialization and more.
